# Max modifier

A modifier which creates a validation rule to ensure that a given value is less or equal to the given input.

## Schemas

- `number`: [Learn more](/schemas/number)
- `string`: [Learn more](/schemas/string)
- `date`: [Learn more](/schemas/date)
- `array`: [Learn more](/schemas/array)

## Arguments

- `max` - The maximum value that the input value must meet or exceed
- `message?` - A custom object with error messages to be displayed when validation fails or there are type errors
