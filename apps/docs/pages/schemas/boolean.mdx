# Boolean schema

A function which creates a parsing rule to ensure that a given value is a boolean.

## Arguments

1. (Optional): A list of modifiers
   - `custom`: [Learn more](/modifiers/custom).
2. (Optional): A custom object with errors messages to be displayed when validation fails or there are type errors.


## Example

```ts
import { boolean } from '@nordic-ui/validathor';

const booleanRule = boolean('Value must be a boolean');

try {
  const parsedValue = booleanRule.parse(true);
  console.log('Parsed value:', parsedValue);
} catch (error) {
  console.error('Parsing failed:', error.message);
}
```
