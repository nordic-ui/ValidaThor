# Date schema

A function which creates a parsing rule to ensure that a given value is a date.

## Arguments

1. (Optional): A list of modifiers
    - `min`: [Learn more](/modifiers/min).
    - `max`: [Learn more](/modifiers/max).
    - `custom`: [Learn more](/modifiers/custom).
2. (Optional): A custom object with error messages to be displayed when validation fails or there are type errors.

## Example

```ts
import { date } from '@nordic-ui/validathor';

const dateRule = date(undefined, 'Value must be a date');

try {
  const parsedValue = dateRule.parse(new Date())
  console.log('Parsed value:', parsedValue);
} catch (error) {
  console.error('Parsing failed:', error.message)
}
```
